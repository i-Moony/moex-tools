## 1. Получение ежедневных свечей разных типов инструментов за определенный интервал

Скрипт:

1. Получает ежедневные данные для каждого из инструментов за указанный период с биржи MOEX.
2. Заполняет отсутствующие даты либо пустыми значениями, либо предыдущими значениями (если указан параметр "fill")
3. Объединяет таблицы по одному параметру (по умолчанию "close" - цена на закрытие торговой сессии, за это отвечает параметр "base")

По итогу получается очень удобная табличка для бек-тестов или построения эконометрических моделей.

### Пример выходных данных

```tty
            IMOEX_close  SBER_close  MOEX_close  T_close
begin
2022-01-01      3852.50      305.47      156.46   6364.4
2022-01-02      3852.50      305.47      156.46   6364.4
2022-01-03      3852.50      305.47      156.46   6364.4
2022-01-04      3873.49      304.24      155.95   6264.6
2022-01-05      3815.05      287.01      148.24   5805.0
```

Содержимое CSV-файла:

```csv
begin;IMOEX_close;SBER_close;MOEX_close;T_close
2022-01-01;3852,5;305,47;156,46;6364,4
2022-01-02;3852,5;305,47;156,46;6364,4
2022-01-03;3852,5;305,47;156,46;6364,4
2022-01-04;3873,49;304,24;155,95;6264,6
2022-01-05;3815,05;287,01;148,24;5805,0
```

### Пример входных данных

```txt
2022-01-01 2026-01-01
индекс IMOEX
акция sber
акция moex
акция t
```

### Help

```tty
usage: index.py [-h] [-i INPUT] [-o OUTPUT] [-f] [-b BASE]

Получение данных дневных свечей с MOEX ISS в удобном для эконометрики формате

options:
  -h, --help           show this help message and exit
  -i, --input INPUT    Файл со списком необходимых бумаг (по умолчанию: input.txt)
  -o, --output OUTPUT  Файл, в который будут выведены данные (по умолчанию: output.csv)
  -f, --fill           Заполнять ли дни без сделок значениями (по умолчанию: нет)
  -b, --base BASE      Какой параметр оставить для сравнения (по умолчанию: close) (может быть: close, open, high, low, volume, value)
```

### Текущие ограничения

Пока не получается "вытащить" данные по денежным фондам.
